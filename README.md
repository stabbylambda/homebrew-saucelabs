homebrew-saucelabs
===============
This tap provides a quick way to install sauce-connect from Saucelabs.

How do I install this formula?
--------------------------------
Just `brew tap stabbylambda/saucelabs` and then `brew install
sauce-connect`.

`brew help`, `man brew`, or the Homebrew [wiki][].
